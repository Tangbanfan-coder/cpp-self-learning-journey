# 这是我在bilibili上看黑马程序员的C++课程所写的一个阶段性案例，一个基于控制台的通讯录管理系统

## 主要功能：

1.添加联系人信息—— 录入姓名、性别、年龄、电话、住址

2.显示所有联系人—— 以表格形式列出所有联系人信息

3.删除指定联系人 —— 按姓名删除指定联系人

4.按姓名查找联系人— 按姓名精确查找并显示详细信息

5.修改联系人信息—— 重新输入指定联系人的全部信息

6.一键清空通讯录—— 一键清空，操作前二次确认

7.退出系统—— 按 0 退出并显示告别语

注意：当前版本数据仅保存在内存中，关闭程序后数据不保留。

## 项目	说明
语言	C++（C++11 或更高，兼容 C++98）

编译器	Visual Studio 2022（MSVC）

操作系统	Windows（使用了 system("pause") 和 system("cls")）

依赖	无任何第三方库，仅标准库 iostream string

如果换用 g++/MinGW 编译，同样可以运行（Windows 下需要保留 system 调用）。

## 交互示范：

输入 1，依次填写姓名、性别、年龄、电话、住址，回车即完成添加

输入 2，显示所有联系人表格

输入 3，再输入要删除的联系人姓名，即可删除（按姓名匹配）

输入 4，输入姓名进行精确查找

输入 5，输入姓名后，重新输入全部信息完成修改

输入 6，程序会要求输入 y 确认清空，输入 n 返回

输入 0，退出程序，显示“欢迎再次使用本系统!”

每步操作后按任意键返回主菜单（清屏效果）。
